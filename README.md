# Event Management Web App

A simple Event Management Web Application built with ReactJS that allows users to create, view, and manage events through an interactive and responsive interface.

This project demonstrates the use of React components, state management, and modern frontend development practices.

## Features

- Create and manage events
- Add event details such as title, date, and description
- View a list of upcoming events
- Delete events when no longer needed
- Responsive user interface
- Built using reusable React components

## Tech Stack

**Frontend**

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## Installation & Setup

1. Clone the repository
   git clone https://github.com/anachauhan/Event-Management-ReactJS.git
2. Navigate to the folder
   cd Event-Management-ReactJS
3. Install dependencies
   npm install
4. Run the project
   npm start
